# Assets для лендінгу Scoopy Log

## Необхідні зображення

### 1. Hero Illustration
**Файл:** `hero-illustration.png`  
**Розмір:** Рекомендовано 600x600px або більше  
**Опис:** Ілюстрація щасливої дитини або сім'ї (можна використати placeholder)

### 2. Screenshots додатку
**Файли:**
- `screenshot-sleep.png` - екран відстеження сну
- `screenshot-feeding.png` - екран відстеження годування
- `screenshot-diapers.png` - екран відстеження підгузників

**Розмір:** Рекомендовано 375x812px (iPhone розмір) або пропорційно  
**Опис:** Реальні скріншоти з iOS додатку в світлій темі

### 3. Open Graph Image
**Файл:** `og-image.png`  
**Розмір:** 1200x630px  
**Опис:** Зображення для соціальних мереж (Facebook, Twitter)

## Placeholder

Якщо зображення ще не готові, можна використати placeholder сервіси:
- https://via.placeholder.com/600x600
- https://placehold.co/600x600

Або створити простий SVG placeholder в HTML.

## Оптимізація

Перед завантаженням оптимізуйте зображення:
- Використовуйте WebP формат для кращої продуктивності
- Стисніть PNG/JPG через TinyPNG або подібні сервіси
- Додайте `loading="lazy"` для зображень нижче fold
